# StudGen
Программа получает на вход датасет с информацией об идентификаторе учащихся; количестве часов, потраченных на учёбу; количество часов, потраченных на сон; проценте посещаемости и итоговых баллах на экзамене и выдает отчет о зависимостях между итоговой оценкой и другими показателями, а также о 10 наиболее и наименее успешных студентах и о привычках, приведших их к соответствующим результатам. Поимо этого программа составляет рекомендацию о том, сколько времени нужно было уделять каждой категории, чтобы попасть в топ студентов.

Пункты проделанные в качестве индивидуальных правок:
1) Исправлпены недочеты на которые было указано при сдачи проекта:
   \n1.1 В парсе убран айди студентов и добавлина проверка на наличие пустых строк, а не просто их удаление(в нашем случае их нет).
   1.2 В модуле с созданием графиков для некоторых столбцов тип графика был изменен на box plot и теперь графики загружаються в отдельную папку.
   1.3 Был переработан модуль top теперь он выдает базовую статистику (макс, мин, среднее и медиану) для первых скольки то учеников (сколько можно выбирать) поэтому теперь функционал сильно отличается от pattern.
   1.4 Добавленна точка входа name == main.

2) Модуль predict:
   при изучении графиков я заметил, что есть кореляция между столбцами hours_studied, attendance и оценкой за экзамен, поэтому я решил попробовать с максимальной точностью на их основе предсказать оценку.
   Прорамма сначала вычисляет средний импакт еденицы из выбраных столбцов на оценку. Далее для пар коофицентов i и 100 - i(чтобы итоговые коофиценты были до 2 знаков после запятой) ищет среднюю погрешность,
   ТК функция  (hours_studied*i + attendance * (100 - i) имеет один экстренум, то мы можем просто пройтись за линию по значениям, и как только предыдущая погрешность будет меньше новой, значит мы нашли итоговый результат
